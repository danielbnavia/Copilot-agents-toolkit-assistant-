# Meeting Insights Bot

Extract action items, summaries, and insights from Teams meetings.

## Features

- Real-time meeting transcription
- Action item extraction
- Meeting summary generation
- Participant insights
- Follow-up reminders
- Integration with task management

## Capabilities

- Teams meeting API
- Microsoft Graph for calendar
- AI summarization
- Task creation in Planner/To Do

## Setup

1. Configure Teams app permissions
2. Enable meeting transcription
3. Set up AI models for summarization
4. Deploy to Teams

## Conversation Starters

- "Summarize my last meeting"
- "What are my action items?"
- "Schedule a follow-up"
- "Show meeting insights for this week"
