# HR Onboarding Bot

Automated employee onboarding assistant that streamlines the new hire process.

## Features

- Welcome new employees
- Distribute onboarding materials
- Schedule orientation sessions
- IT equipment requests
- Compliance training tracking
- Team introductions

## Capabilities

- Microsoft Graph for user provisioning
- SharePoint for document management
- Teams for channel creation
- Power Automate for workflow automation

## Setup

1. Configure Graph API permissions for user management
2. Set up SharePoint document library for onboarding materials
3. Create Teams template for new employee channels
4. Deploy bot to company Teams environment

## Conversation Starters

- "Start onboarding for new employee"
- "Check onboarding progress for [name]"
- "Send welcome package"
- "Schedule orientation sessions"
