# Sales Assistant Agent

An intelligent sales assistant that helps with lead qualification, CRM integration, and sales pipeline management.

## Features

- Lead qualification based on BANT criteria
- CRM integration (Dynamics 365, Salesforce)
- Meeting scheduling and follow-ups
- Sales analytics and reporting
- Quote generation

## Capabilities

- Microsoft Graph API for calendar and email
- Custom API for CRM integration
- Web search for company research
- Adaptive cards for interactive responses

## Setup

1. Copy files to your project directory
2. Update the API plugin configuration with your CRM credentials
3. Deploy to Teams or use in Copilot Studio

## Conversation Starters

- "Qualify this lead: [company name]"
- "Schedule a follow-up with [contact]"
- "Show me my sales pipeline"
- "Generate a quote for [product]"
