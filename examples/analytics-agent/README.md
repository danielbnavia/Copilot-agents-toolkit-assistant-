# Data Analytics Agent

Intelligent data analytics assistant for querying, visualizing, and deriving insights from business data.

## Features

- Natural language data queries
- Automated report generation
- Data visualization
- Trend analysis
- Predictive insights
- Dashboard creation

## Capabilities

- Power BI integration
- Azure SQL Database queries
- Excel data analysis
- Custom API for data sources
- AI-powered insights

## Setup

1. Configure database connections
2. Set up Power BI workspace
3. Define data schemas and permissions
4. Deploy to Teams or Copilot Studio

## Conversation Starters

- "Show sales performance for Q4"
- "What are the top trends in customer data?"
- "Generate monthly revenue report"
- "Compare year-over-year growth"
