### What are M365 Agents? (0:30-2:30)
"Microsoft 365 agents are AI-powered assistants that integrate directly into your workflow. They can:"

[Show visual examples]
- Answer questions using your organization's data
- Automate repetitive tasks
- Provide personalized recommendations
- Integrate with existing Microsoft 365 services

"The beauty of declarative agents is that you define WHAT you want the agent to do, not HOW to do it. This makes them incredibly powerful yet easy to build."

### Use Cases (2:30-5:00)

### Architecture Overview (5:00-7:30)
"How do these agents work? Let's break it down:"

[Show architecture diagram]

1. **Declarative Manifest**: JSON file defining the agent
2. **Capabilities**: What the agent can access (Web, Graph, etc.)
3. **Instructions**: How the agent should behave
4. **Integrations**: Teams, Copilot, etc.
