# Video Tutorials - M365 Agent Toolkit

Professional video tutorial series for building Microsoft 365 agents.

## Tutorial Series

### Getting Started (Beginner)
1. **Introduction to M365 Agents** (10 min)
   - What are declarative agents?
   - Use cases and benefits
   - Toolkit overview

2. **Your First Agent** (15 min)
   - Creating a simple agent
   - Understanding the manifest
   - Testing in Teams

3. **Agent Capabilities** (12 min)
   - Web Search
   - Microsoft Graph
   - OneDrive & SharePoint
   - Graph Connectors

### Intermediate
4. **Building a Customer Support Agent** (25 min)
   - Requirements gathering
   - Designing conversation flow
   - Implementing capabilities
   - Testing and iteration

5. **API Plugins** (20 min)
   - Creating OpenAPI specs
   - Authentication (OAuth 2.0, API keys)
   - Response semantics
   - Testing plugins

6. **Adaptive Cards** (18 min)
   - Card designer overview
   - Interactive elements
   - Actions and submissions
   - Best practices

### Advanced
7. **Teams Bot Development** (30 min)
   - Bot Framework SDK
   - Conversational bots
   - Meeting bots
   - Notification bots

8. **Workflow Automation** (25 min)
   - Power Automate integration
   - Approval flows
   - Scheduled automation
   - Error handling

9. **Azure AI Integration** (30 min)
   - Azure OpenAI setup
   - RAG with AI Search
   - Custom models
   - Multimodal capabilities

10. **Copilot Studio Integration** (25 min)
    - Low-code agent building
    - Custom engines
    - Topic management
    - Enterprise deployment

### Professional
11. **Production Deployment** (35 min)
    - CI/CD pipelines
    - Testing strategies
    - Monitoring and logging
    - Security best practices

12. **Enterprise Scale** (30 min)
    - Multi-tenant architecture
    - Performance optimization
    - Governance and compliance
    - Cost management

## Production Setup

### Equipment Needed
- Screen recording software (OBS Studio, Camtasia)
- Microphone (Blue Yeti, Rode NT-USB)
- Video editing software (Adobe Premiere, DaVinci Resolve)
- Thumbnail creation (Canva, Photoshop)

### Recording Guidelines
1. **Preparation**
   - Write detailed script
   - Prepare demo environment
   - Test all code/examples
   - Create slide deck

2. **Recording**
   - 1080p or 4K resolution
   - Clear audio (test levels)
   - Consistent pacing
   - Show keyboard shortcuts

3. **Editing**
   - Add intro/outro
   - Insert callouts and highlights
   - Background music (optional)
   - Chapter markers

4. **Publishing**
   - Upload to YouTube/Stream
   - Add chapters and timestamps
   - Create transcript
   - SEO optimization (title, tags, description)
